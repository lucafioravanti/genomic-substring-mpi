# Substring Matching

This project performs substring matching on large genomic sequences using serial, OpenMP, and MPI implementations in C.

## How to Use

### Compile the Project

To compile all three versions, simply run:

```bash
make
```

This will compile the following:
- `src/main_serial.c` → creates the executable `serial`
- `src/main_parallel.c` → creates the executable `parallel_omp` (uses OpenMP)
- `src/main_parallel_mpi.c` → creates the executable `parallel_mpi` (uses MPI)

---

### Run the Serial Version

Once compiled, you can run the serial version like this:

```bash
./serial data/test_dna.txt ACGTA
```

This command will:
- Read the DNA sequence from `data/test_dna.txt`
- Search for the substring `"ACGTA"`
- Print the positions where the pattern occurs and execute the Python analysis script to plot performance graphs.

---

### Run the OpenMP Version

To run the OpenMP version:

```bash
./parallel_omp data/test_dna.txt ACGTA
```

---

### Run the Parallel Version (MPI)

To run the MPI version, use `mpiexec` and specify the number of processes (e.g., 4):

```bash
mpiexec -n 4 ./parallel_mpi data/test_dna.txt ACGTA
```

This command will:
- Launch 4 MPI processes
- Each process searches a portion of the sequence
- Rank 0 will consolidate and report the matching count

---

### Note

Make sure OpenMP and MPI are installed on your system. You will also need Python installed with `pandas` and `plotly` to view the graphical reports automatically generated by the serial run.
